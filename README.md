# algorithms-mipt

В этом репозитории я собираю свои решения алгоритмических задач.

## Структура репозитория

Код разбит по темам и паттернам для удобной навигации:

- `/dp` — Задачи на динамическое программирование
- `/graphs` — Обход в ширину (BFS), глубину (DFS), кратчайшие пути
- `/flows and matchings` — Задачи на паросочетания и потоки
- `/simple algos` — Простые задачи
- `/trees` — задачи на различные виды деревьев (Splay, AVL, КЧД, ДО, Дерево Фенвика итд.)
